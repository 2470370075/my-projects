### 冒泡排序
冒泡：两个相比，大的往后，大的为泡，冒到最后
```
l = [2,5,9,2,4,6,8,9,2]

def bubble_sort(l):
    n = len(l)
    for i in range(n-1,0,-1):
        c = 0
        for j in range(0,i):
            if l[j] > l[j+1]:
                c += 1
                l[j+1],l[j] = l[j],l[j+1]
        if c == 0:
            return l

print(bubble_sort(l))
```

插入：后面部分里的第一个，插入到前面部分合适的位置
```
def insert_sort(l):
    n = len(l)
    for i in range(1,n):
        for j in range(i,0,-1):
            if l[j] < l[j-1]:
                l[j],l[j-1] = l[j-1],l[j]
            else:
                break
    return l
```
